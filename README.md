# Capstone-project-A-SHIMA
## 概要
- ドライブの目的地決定のためのアプリケーション
- 時間、料金を入れてそれをもとにいける観光地を地図に可視化
- 追加機能として、行き先の観光地から行ける表示できるようにしておく

## ディレクトリ構造
```
.
├── README.md
├── backend
|   ├── data
├── frontend
```

## ビルド方法
```
#クローン
git clone 
cd DSN

#フロント
cd frontend
make install
make

# バックエンド（別ターミナルを立ち上げ）
cd backend
make install
make

localhost:3000にアクセス


```
